生成随机字符串
    [Documentation]    先选定一个string的常量，然后判断他的长度，设定要生成多少位长度的字符串，然后使用循环生成我需要的随机字符串
    #获取ASCII字符串集，大小写字母集，共52个字符
    ${str}    Evaluate    string.ascii_letters    string
    ${len}    Evaluate    len('${str}')    #52
    ${num}    Set Variable    10
    ${newstr}    Set Variable    ${EMPTY}
    :FOR    ${index}    IN RANGE    ${num}
    \    ${i}    Evaluate    random.randint(0,int(${len})-1)    random    #0~52间的随机数
    \    ${tmp}    Set Variable    ${str[int(${i})-1]}
    \    ${newstr}    Set Variable    ${newstr}${tmp}
    log    ${newstr}
